chicken-pickle


Developed a full-stack web application for online chicken pickle orders with a UPI payment gateway. The project features a responsive interface with HTML, CSS, and JavaScript for dynamic content and user interactions. It includes pages for order placement, payment processing, and address capture, with successful or failed transaction redirects. Built with Node.js for backend operations, the app demonstrates skills in both front-end and back-end development, offering a seamless user experience and secure payment handling.

OnlyRace


Developed a multimedia-rich website, designed to engage users with curated video content across various genres, including Epic, Commercial, Religious, LGBTQ, Women's Issues, and Ethnicity. The site features a dynamic search function and a CSS-styled dropdown menu for intuitive navigation. A scrolling marquee encourages users to support the project by purchasing premium content through a "Buy Me a Coffee" initiative, with instructions for UPI payments and WhatsApp communication. The website is built using HTML, CSS, and JavaScript, demonstrating proficiency in front-end web development and user experience design.
